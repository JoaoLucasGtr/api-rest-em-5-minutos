## api-rest-em-5-minutos

Exemplo de criação de uma api rest simples

### Requerimentos:
  * Node / Npm;
  * MongoDb;
  
### Install
  * npm install;
  * npm install -g nodemon;
  
### Run
  
  $ nodemon server.js
